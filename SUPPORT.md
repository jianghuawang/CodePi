# Support

CodePi is a community-maintained open-source project.

- Use the structured [issue forms](https://github.com/jianghuawang/CodePi/issues/new/choose)
  for setup questions, reproducible bugs, and feature requests.
- Use private vulnerability reporting as described in [SECURITY.md](SECURITY.md)
  for security-sensitive reports.

Please include your CodePi version, macOS version, Node.js version, Pi version,
and concise reproduction steps. Support is best-effort and has no guaranteed
response time.
